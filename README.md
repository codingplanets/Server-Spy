## Getting Started

You will need a few different modules installed to execute Server-Spy.



## Install Server-Spy

```
git clone https://github.com/codingplanets/Server-Spy
```
```
cd Server-Spy
```
```
chmod +x *
```
## Usages
```
usage: ./Server-Spy.py
```
